# EasyFood 🍽️

## Descripción del Proyecto

EasyFood es una plataforma web desarrollada como proyecto para el ramo de Innovación y Emprendimiento. Su objetivo es ayudar a las personas a encontrar lugares donde comer de manera rápida, sencilla y personalizada.

Muchas veces los usuarios no saben qué restaurante elegir debido a la gran cantidad de opciones disponibles. EasyFood busca resolver este problema mediante recomendaciones basadas en las preferencias del usuario, ubicación, tipo de comida, presupuesto y valoraciones de otros clientes.

## Problema que Resuelve

Elegir dónde comer puede ser una tarea complicada y consumir tiempo. Los usuarios suelen revisar múltiples aplicaciones, sitios web y redes sociales antes de tomar una decisión.

EasyFood centraliza esta información en una sola plataforma para facilitar la elección y mejorar la experiencia del usuario.

## Objetivos

* Facilitar la búsqueda de restaurantes y locales de comida.
* Reducir el tiempo de decisión al elegir dónde comer.
* Entregar recomendaciones personalizadas.
* Mostrar información relevante como ubicación, precios, horarios y valoraciones.
* Mejorar la experiencia gastronómica de los usuarios.

## Funcionalidades

* 🔍 Búsqueda de restaurantes.
* 📍 Filtrado por ubicación.
* 🍔 Filtrado por tipo de comida.
* 💲 Filtrado por rango de precios.
* ⭐ Sistema de valoraciones y reseñas.
* ❤️ Guardado de favoritos.
* 🤖 Recomendaciones personalizadas.
* 📱 Diseño adaptable para dispositivos móviles.

## Público Objetivo

EasyFood está dirigido a:

* Estudiantes universitarios.
* Trabajadores que buscan opciones cercanas.
* Familias.
* Turistas.
* Cualquier persona que desee encontrar rápidamente un lugar para comer.

## Propuesta de Valor

EasyFood permite descubrir restaurantes de forma rápida y eficiente, entregando recomendaciones personalizadas que ayudan a los usuarios a tomar mejores decisiones y ahorrar tiempo.
